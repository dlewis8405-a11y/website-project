# D GAMES Website

This repository contains a simple static website for D GAMES.

## Publish on GitHub Pages

1. Commit your changes and push them to GitHub:
   ```bash
git add .
git commit -m "Prepare site for public deployment"
git push origin main
```
2. Open your repository on GitHub.
3. Go to **Settings** > **Pages**.
4. Under **Source**, select the `main` branch and the `/ (root)` folder.
5. Save the settings.

Your site should appear at:

`https://dlewis8405-a11y.github.io/website-project/`

## Notes

- The site now includes metadata for search engines and social sharing.
- `robots.txt` allows search engines to crawl the site.
- `sitemap.xml` helps search engines discover the homepage.
